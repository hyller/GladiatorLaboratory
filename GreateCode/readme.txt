介绍###############################################################################GLADIATOR CODE，中文名角斗士编码，是一款代码美化的软件，由GLADIATOR FIRMWARE（角斗士固件）推出。基于GREAT CODE开源软件。GREAT CODE的配置命令多大150条，而且教程不全面，用户要先花大量时间学习才能使用该软件。GLADIATOR CODE提供经过验证的配置文件，而且美化后的文件遵循常用的规范。初次提供的配置文件美化后代码遵循MISRA-C：2004规范。后续推出更多。提供反馈请发邮件到gladiator.firmware@gmail.com。################################################################################                                1 开始体验###############################################################################本体验目标是美化GladiatorCode自带的一个用于测试的源文件1) 把GLADIATOR CODE解压到D盘，D盘将出现下列目录：GladiatorCode2）去D:\GladiatorCode\Test目录下查看需要美化的源代码3）打开命令行，移动到目录D:\GladiatorCode下4）输入命令 gc -dir-"D:\GladiatorCode\Test"5）去D:\GladiatorCode\Test目录下查看美化结果，原文件会以后缀.bak文件保存################################################################################                                2 初级使用###############################################################################1）移动GladiatorCode根目录到你想存放的地方2）设置PATH，在环境变量中，把PATH设置为GladiatorCode根目录。3）用命令gc -dir-"your source path"美化你的代码4）阅读gc.cfg文件，可以作一些简单的配置更改################################################################################                                3
