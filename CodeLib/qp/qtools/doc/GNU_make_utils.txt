About GNU_make_utils
====================
This distribution contains the GNU make 3.82 and other related UNIX-style file
utilities, such as cp, mkdir, mv, mvdir, rm, rmdir, and touch, which are commonly
used in the makefiles. The utilities are compiled to run natively on Windows
without any additional DLLs.

http://prdownloads.sourceforge.net/mingw/mingw32-make-3.81-1.tar.gz

http://prdownloads.sourceforge.net/mingw/mingw32-make-3.81-1-src.tar.gz

http://www.gnu.org/software/make

The file utilities source (source/fileutils-3.16-src.zip) has been taken from: 

http://prdownloads.sourceforge.net/unxutils/UnxUtilsSrc.zip
